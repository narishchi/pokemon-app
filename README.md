# Pokemon App

เว็บแอปพลิเคชันแสดงข้อมูลโปเกม่อนพัฒนาด้วย Next.js ร่วมกับ Material-UI (MUI) และเชื่อมต่อข้อมูลโดยตรงจาก PokeAPI ซึ่งโปรเจกต์นี้เป็นส่วนหนึ่งของรายวิชาFront-end Web Programming

---

## ฟีเจอร์เด่นของโปรเจกต์

### หน้าแรก
* มีการดึงรายชื่อโปเกมอนทั้งหมด1351 ตัว จาก PokeAPI 
*แสดงผลการ์ดรายชื่อ ปรับขนาดตามหน้าจอของอุปกรณ์ที่ใช้งาน

### หน้ารายละเอียดโปเกมอน
* เมื่อกดที่โปเกม่อนแต่ละตัวให้เปิดหน้าใหม่ เพื่อแสดงข้อมูลรายละเอียดของโปเกม่อน อย่างน้อยต้องมี ชื่อ รูป สถานะ วิวัฒนาการ ประเภทของโปเกม่อน เสียงของโปเกม่อน

### about this project 
* ข้อมูลของผู้พัฒนา รายวิชา หลักสูตร มหาวิทยาลัย และลิงค์ไปยัง GitHub Source code
